Preparacion Basica del ambiente para desarrolar  

1. Instalar (verificar dotnet --version)  .netCore sdk  3.1.101  ubicada  en  \\10.10.1.196\installs\.NET Core 
	o obtenerla  del sitio odicial de microsoft
2. Verificar  version de visula studio 16.4.X
3. Tener instalado  Visual Studio CODE para codificacion  del front  
	este puede ser cualquier  editor de texto a su prefetencia  
4. Instalar  Node js ultima version  (node --version)
5. Instalar GitHub Desktop.
